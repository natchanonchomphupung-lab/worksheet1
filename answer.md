# แบบฝึกหัด Git

ชื่อ-นามสกุล : ณัฐชนน ชมภูพวง

ห้อง : 2

เลขที่ : 4

----------------------------------

ข้อที่ 1
Git คืออะไร

ตอบ : ระบบที่ใช้ติดตามการเปลี่ยนแปลงของไฟล์

----------------------------------

ข้อที่ 2
Repository คืออะไร

ตอบ : ที่เก็บงานหรือแชร์ไฟล์งานต่างๆ

----------------------------------

ข้อที่ 3
คำสั่งใดใช้ Clone Repository

ตอบ : git clone <Repo>

----------------------------------

ข้อที่ 4
คำสั่งใดใช้เพิ่มไฟล์เข้าสู่ Staging Area

ตอบ :git add <file>

----------------------------------

ข้อที่ 5
คำสั่งใดใช้บันทึกการเปลี่ยนแปลง

ตอบ : git commit -m "คอมเมนต์"

----------------------------------

ข้อที่ 6
คำสั่งใดใช้ส่งข้อมูลขึ้น GitHub

ตอบ : git push

----------------------------------

ข้อที่ 7
วันนี้ได้เรียนรู้อะไรเกี่ยวกับ Git

ตอบ :คำสั่งclone
